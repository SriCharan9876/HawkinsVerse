HawkinsVerse is a full-stack web application built using the **MERN** stack. It combines a visually improved frontend with a robust backend to deliver features like secure authentication, character galleries, and an interactive chat system.

## ‚ú® Features

-   **Stranger Things Aesthetic**: A completely themed UI with backgrounds, fonts, and styles inspired by the show.
-   **User Authentication**:
    -   Secure **Sign Up** and **Login** using JWT (JSON Web Tokens).
    -   **OTP Verification** via Email (Nodemailer) during registration.
    -   **Google OAuth** integration for one-click sign-in.
-   **Character Gallery**: Browse through profiles of your favorite characters from the series.
-   **Interactive Chat**:
    -   Chat with characters from the show.
    -   **User-Specific History**: Your conversations are saved privately, so you can pick up where you left off.
-   **Responsive Design**: Optimized for different screen sizes, ensuring a seamless experience on both desktop and mobile.

### **Frontend**
-   **React** (Vite): Fast and modern UI library.

-   **JWT (JSON Web Tokens)**: For secure, stateless authentication.

-   **Google Auth Library**: Backend verification for Google Sign-In.
-   **BCrypt**: Passwords hashing for security.
